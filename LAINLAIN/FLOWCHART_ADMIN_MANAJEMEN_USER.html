Penjelasan Flowchart Admin - Manajemen User

Flowchart ini menggambarkan alur manajemen user oleh admin dalam sistem rank checker. Proses dimulai ketika admin mengakses menu manajemen user melalui dashboard admin. Admin kemudian melihat daftar semua user yang terdaftar dalam sistem beserta informasi lengkap seperti username, email, WhatsApp, role, dan status aktif. Setelah melihat daftar, admin dapat memilih aksi yang akan dilakukan terhadap user tertentu.

Admin memiliki empat opsi operasi yaitu update user untuk mengubah data user seperti username, email, WhatsApp, dan role, delete user untuk menghapus user beserta semua data terkait, activate user untuk mengaktifkan akun user yang dinonaktifkan, dan deactivate user untuk menonaktifkan akun user yang aktif. Setelah admin memilih aksi, sistem akan melakukan operasi yang sesuai seperti update data, hapus data, aktifkan akun, atau nonaktifkan akun sesuai dengan pilihan admin.
