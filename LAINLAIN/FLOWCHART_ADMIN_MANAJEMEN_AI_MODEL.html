Penjelasan Flowchart Admin - Manajemen AI Model Config

Flowchart ini menggambarkan alur manajemen konfigurasi model AI oleh admin dalam sistem rank checker. Proses dimulai ketika admin mengakses menu manajemen AI model config melalui dashboard admin. Admin kemudian melihat daftar semua konfigurasi model AI yang tersedia, termasuk informasi seperti model name, display name, quota cost, dan status aktif. Setelah melihat daftar, admin dapat memilih untuk menambahkan model baru, mengupdate konfigurasi model yang sudah ada, atau menghapus model yang tidak diperlukan lagi.

Jika admin memilih untuk menambah model, admin akan mengisi form dengan data model name, display name, quota cost, dan API key, kemudian sistem akan memvalidasi bahwa model name harus unik sebelum menyimpan model baru tersebut. Jika admin memilih untuk update model, admin akan mengedit konfigurasi model seperti model name, quota cost, API key, atau status aktif, kemudian sistem akan memvalidasi model name unik dengan mengecualikan model yang sedang diedit sebelum menyimpan perubahan. Jika admin memilih untuk menghapus model, sistem akan menghapus model tersebut dari database. Setelah operasi selesai, perubahan akan langsung terlihat di daftar AI models.
