📋 Penjelasan Activity Diagram: Cek Rank Website

1. Overview

Activity Diagram ini menggambarkan alur proses pengecekan ranking website di Google Search Engine Results Page (SERP) menggunakan integrasi SerpAPI. Proses ini melibatkan tiga aktor: User, System, dan SerpAPI.

2. Alur Proses

A. Validasi

  • User submit form → System cek UserStatus dan kuota
  • Jika tidak valid atau kuota habis: Tampilkan error, proses berakhir

B. Request API

  • System request ke SerpAPI → SerpAPI proses pencarian Google
  • SerpAPI kembalikan response → System parse dan cari domain

C. Penyimpanan & Tampilan

  • System simpan ke RankResult dan update kuota (-1)
  • System tampilkan hasil → User lihat hasil
