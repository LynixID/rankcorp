📋 Penjelasan Activity Diagram: Pembelian Kuota

1. Overview

Activity Diagram ini menggambarkan alur proses pembelian kuota dengan dua metode pembayaran: Manual Transfer (butuh approval admin) dan Midtrans Payment Gateway (otomatis). Proses ini melibatkan empat aktor: User, System, Admin, dan Midtrans.

2. Alur Proses - Manual Transfer

  • User transfer bank dan upload bukti → System buat QuotaPurchase (Pending)
  • Admin review dan approve → System update kuota dan status premium
  • Jika reject: Proses berakhir

3. Alur Proses - Midtrans

  • User bayar di Midtrans → System terima callback
  • Jika settlement: System update kuota, status premium, dan kirim notifikasi WhatsApp
  • Jika tidak settlement: Proses berakhir
