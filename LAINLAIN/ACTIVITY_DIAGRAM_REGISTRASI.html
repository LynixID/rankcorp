📋 Penjelasan Activity Diagram: Registrasi & OTP Verification

1. Overview

Activity Diagram ini menggambarkan alur proses registrasi pengguna baru dengan sistem verifikasi OTP melalui WhatsApp. Proses ini melibatkan dua aktor utama: User dan System.

2. Alur Proses

A. Tahap Input & Validasi

  • User isi form dan submit → System validasi data
  • Jika tidak valid: Tampilkan error, proses berakhir

B. Tahap OTP

  • System generate OTP dan kirim via WhatsApp → User masukkan OTP
  • System verifikasi OTP
  • Jika salah: Tampilkan error, user dapat coba lagi

C. Tahap Pembuatan Akun

  • Jika OTP benar: System buat User dan UserStatus (3 kuota gratis)
  • User redirect ke login
