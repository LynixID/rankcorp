Penjelasan Flowchart Admin - Manajemen Paket Kuota

Flowchart ini menggambarkan alur manajemen paket kuota oleh admin dalam sistem rank checker. Proses dimulai ketika admin mengakses menu manajemen paket kuota melalui dashboard admin. Admin kemudian melihat daftar semua paket kuota yang tersedia dalam sistem, termasuk informasi seperti nama paket, harga, dan jumlah kuota yang diberikan. Setelah melihat daftar, admin dapat memilih untuk menambahkan paket baru, mengupdate paket yang sudah ada, atau menghapus paket yang tidak diperlukan lagi.

Jika admin memilih untuk menambah paket, admin akan mengisi form dengan data nama paket, harga, dan jumlah kuota, kemudian sistem akan menyimpan paket baru tersebut ke database. Jika admin memilih untuk update paket, admin akan mengedit data paket yang sudah ada seperti nama, harga, atau jumlah kuota, kemudian sistem akan menyimpan perubahan tersebut. Jika admin memilih untuk menghapus paket, sistem akan menghapus paket tersebut dari database. Setelah operasi selesai, perubahan akan langsung terlihat di daftar paket kuota.
