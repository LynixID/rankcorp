📋 Penjelasan Activity Diagram: SEO Analysis dengan AI

1. Overview

Activity Diagram ini menggambarkan alur proses analisis SEO menggunakan AI untuk menganalisis data ranking website. Fitur ini hanya tersedia untuk user dengan status Premium dan menggunakan kuota sesuai model AI yang dipilih. Proses ini melibatkan tiga aktor: User, System, dan AI API (Gemini/OpenAI).

2. Alur Proses

A. Validasi

  • User pilih model AI dan mulai analisis → System validasi status premium dan kuota
  • Jika tidak valid: Tampilkan error, proses berakhir

B. Request AI

  • System analisis rank history dan buat prompt → Request ke AI API
  • AI API generate analisis → System parse dan validasi response

C. Penyimpanan & Tampilan

  • Jika valid: System simpan ke SeoAnalysis dan update kuota
  • System tampilkan hasil → User lihat hasil analisis

3. Model AI yang Tersedia

  • Gemini: 3 kuota - Cepat & murah
  • GPT-3.5: 2 kuota - Stabil, harga menengah
  • GPT-4.1: 5 kuota - Terbaik, analisis mendalam
