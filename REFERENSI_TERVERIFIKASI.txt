APLIKASI CHECK RANKRING

1)	Visual Studio Code
Dikembangkan oleh Microsoft, Visual Studio Code merupakan editor kode sumber lintas platform yang tersedia secara gratis dan open-source untuk sistem operasi Windows, macOS, dan Linux. Editor ini secara native mendukung berbagai bahasa pemrograman seperti JavaScript, TypeScript, dan Node.js, sementara dukungan untuk bahasa pemrograman lainnya seperti C++, C#, Python, Go, Java, dan PHP dapat ditambahkan melalui ekstensi yang tersedia di marketplace Visual Studio Code (Bihesa et al., 2022).

2)	Google Chrome
Sebagai peramban web lintas platform yang dikembangkan oleh Google, Chrome pertama kali dirilis pada tanggal 2 September 2008 dan tersedia untuk berbagai sistem operasi termasuk Windows, macOS, Linux, iOS, dan Android. Peramban ini dikenal dengan performa yang cepat, antarmuka yang minimalis, serta fitur keamanan seperti sandboxing dan pembaruan otomatis yang melindungi pengguna dari ancaman keamanan (Hartanto, 2023).

3)	SQLite3 (via DB Browser for SQLite)
SQLite3 merupakan sistem manajemen basis data relasional yang ringan, serverless, dan mandiri, sering digunakan dalam aplikasi yang memerlukan penyimpanan data lokal tanpa membutuhkan server database terpisah. DB Browser for SQLite menyediakan antarmuka grafis yang memudahkan pengguna dalam membuat, merancang, mengedit, dan memvisualisasikan basis data SQLite tanpa perlu menulis perintah SQL secara manual, tersedia untuk platform Windows, macOS, dan Linux (Khalid & Megawati, 2022).

4)	Draw.io
Draw.io adalah aplikasi diagram berbasis web yang bersifat open-source dan dapat diakses melalui peramban tanpa memerlukan instalasi tambahan. Aplikasi ini memungkinkan pengguna untuk membuat berbagai jenis diagram termasuk diagram alur, diagram jaringan, diagram UML, dan diagram organisasi dengan antarmuka yang intuitif serta integrasi dengan platform penyimpanan cloud seperti Google Drive dan OneDrive (Muthahhari et al., 2021).

5)	Python
Python adalah bahasa pemrograman tingkat tinggi yang ditafsirkan dengan filosofi desain yang menekankan keterbacaan kode dan sintaks yang sederhana. Bahasa ini mendukung berbagai paradigma pemrograman termasuk prosedural, berorientasi objek, dan fungsional, serta memiliki ekosistem pustaka yang luas yang menjadikannya populer untuk pengembangan web, analisis data, kecerdasan buatan, pembelajaran mesin, dan otomatisasi (Ro'if et al., 2024).

6)	JavaScript
Sebagai bahasa pemrograman yang terutama digunakan untuk pengembangan web, JavaScript memungkinkan pembuatan konten interaktif dan dinamis pada halaman web melalui manipulasi Document Object Model (DOM) dan komunikasi asinkron dengan server. Bahasa ini dapat dijalankan di sisi klien maupun server melalui platform seperti Node.js, dan bersama dengan HTML serta CSS membentuk fondasi utama teknologi web modern (Ardiansyah, 2022).

7)	Django
Django adalah framework web berbasis Python yang dirilis di bawah lisensi BSD dan mengikuti pola arsitektur Model-View-Template (MVT) yang memisahkan logika bisnis, tampilan, dan data untuk memudahkan pengelolaan kode. Framework ini dirancang untuk mempercepat pengembangan aplikasi web dengan menyediakan berbagai komponen siap pakai seperti sistem autentikasi pengguna, panel administrasi otomatis, dan ORM (Object-Relational Mapping) yang memudahkan interaksi dengan basis data tanpa perlu menulis kode SQL secara langsung, sehingga pengembang dapat fokus pada logika aplikasi (Akbar et al., 2024).

8)	HTML
HTML (HyperText Markup Language) merupakan bahasa markup standar yang digunakan untuk membuat dan menyusun struktur halaman web dengan menggunakan elemen-elemen seperti tag dan atribut untuk mendefinisikan konten seperti teks, gambar, tautan, dan formulir. Sebagai fondasi utama dalam pengembangan web, HTML berfungsi sebagai kerangka dasar yang menentukan bagaimana informasi ditampilkan di peramban, dan dapat ditingkatkan dengan CSS untuk styling serta JavaScript untuk menambahkan interaktivitas pada halaman web (Wibowo, 2021).

9)	Tailwind CSS
Tailwind CSS mengadopsi pendekatan utility-first dalam framework CSS, di mana pengembang dapat membangun desain yang responsif dan custom dengan cepat melalui penggunaan kelas-kelas utilitas yang dapat dikombinasikan langsung dalam markup HTML tanpa perlu menulis CSS kustom yang ekstensif. Berbeda dengan framework CSS tradisional yang menyediakan komponen siap pakai, Tailwind memberikan fleksibilitas penuh dalam penyesuaian desain dengan menyediakan berbagai utility classes untuk mengatur margin, padding, warna, dan properti lainnya, sehingga memungkinkan prototipe dan iterasi desain yang lebih cepat (Dewi, 2023).

APLIKASI POHON KELUARGA

1)	Visual Studio Code
Visual Studio Code (VS Code) merupakan editor kode modern yang dikenal ringan namun kaya fitur seperti IntelliSense, debugging, dan integrasi Git yang mendukung berbagai bahasa pemrograman melalui ekstensi. Lingkungan kerja yang fleksibel serta antarmuka sederhana membuat VS Code banyak digunakan dalam proses pengembangan aplikasi berskala kecil hingga besar untuk meningkatkan produktivitas pengembang (Bihesa et al., 2022).

2)	Google Chrome
Google Chrome merupakan peramban web yang menawarkan performa tinggi, keamanan kuat, dan kompatibilitas luas terhadap teknologi web modern. Fitur Chrome DevTools membantu pengembang dalam debugging dan pengujian aplikasi secara langsung di browser sehingga mempercepat proses pengembangan web (Hartanto, 2023).

3)	Laragon
Laragon adalah platform pengembangan lokal yang menyediakan paket server lengkap seperti Apache, MySQL/MariaDB, PHP, serta alat bantu seperti Composer dalam satu lingkungan yang ringan dan mudah dikonfigurasi. Dengan fitur auto virtual host dan quick create project, Laragon terbukti meningkatkan kecepatan inisialisasi proyek web berbasis PHP maupun framework Laravel (Suryana, 2021).

4)	PHP
PHP adalah bahasa pemrograman sisi server yang digunakan untuk membangun aplikasi web dinamis dan sistem informasi. Sintaksnya yang mudah dipahami serta dukungan komunitas yang besar membuat PHP tetap menjadi pilihan utama dalam pengembangan web modern, terutama pada sistem yang membutuhkan stabilitas dan skalabilitas (Santoso, 2022).

5)	Laravel
Laravel adalah framework PHP berbasis arsitektur MVC (Model–View–Controller) yang menyediakan berbagai fitur seperti Eloquent ORM, Blade templating, middleware, dan autentikasi bawaan untuk mempercepat proses pengembangan aplikasi. Kemampuannya dalam menghasilkan struktur kode yang rapi menjadikannya salah satu framework paling populer di Indonesia (Hikmah, 2023).

6)	HTML
HTML merupakan bahasa markup standar yang digunakan untuk membangun struktur dasar halaman web dengan menyediakan elemen seperti teks, gambar, tabel, hingga formulir. Sebagai fondasi utama website, HTML berperan penting dalam menentukan bagaimana konten ditampilkan dan berinteraksi dengan CSS maupun JavaScript (Wibowo, 2021).

7)	JavaScript
JavaScript adalah bahasa pemrograman yang memungkinkan halaman web menjadi interaktif melalui manipulasi DOM (Document Object Model), pengolahan event, dan implementasi AJAX. Bahasa ini sangat fleksibel karena dapat berjalan di sisi klien maupun server, sehingga menjadi komponen utama dalam pengembangan web modern (Ardiansyah, 2022).

8)	Tailwind CSS
Tailwind CSS adalah framework CSS berbasis utility-first yang menyediakan kelas-kelas siap pakai sehingga pengembang dapat membangun antarmuka secara cepat tanpa menulis banyak CSS manual. Pendekatan ini menghasilkan proses styling yang lebih efisien serta tampilan antarmuka yang konsisten di berbagai perangkat (Dewi, 2023).

PEMBUATAN ARTIKEL SEO

1)	Google Chrome
Google Chrome digunakan sebagai peramban utama untuk melakukan riset kata kunci, menguji tampilan artikel, serta memastikan struktur SEO seperti meta tag, heading, dan internal link tampil dengan benar di website sebelum dipublikasikan (Hartanto, 2023).

Aplikasi WSP digunakan untuk mengunggah artikel ke website perusahaan, mengatur tata letak konten, dan memastikan gambar serta tautan sesuai standar penulisan, sehingga mempermudah manajemen konten selama proses publikasi (Siregar & Sari, 2021).

3)	AI Studio
AI Studio adalah platform kecerdasan buatan (Artificial Intelligence) yang menyediakan berbagai tools untuk pembuatan konten digital, seperti pembuatan gambar, teks, video, dan model AI lainnya dalam satu lingkungan terpadu. Platform ini digunakan untuk membantu produksi konten otomatis, meningkatkan efisiensi, serta mempercepat proses kreatif tanpa perlu kemampuan teknis yang kompleks. Menurut Susanto (2022), penggunaan AI Studio terbukti mempermudah proses pembuatan konten karena didukung oleh antarmuka intuitif dan kemampuan AI generatif yang stabil.

Google Chrome merupakan peramban web yang dirancang untuk memberikan performa tinggi, stabilitas kuat, dan kompatibilitas penuh terhadap teknologi web modern seperti HTML5, CSS3, dan JavaScript. Chrome banyak digunakan dalam pengembangan web karena menyediakan fitur seperti Chrome DevTools yang memungkinkan inspeksi elemen, debugging, analisis performa, serta pengujian responsivitas secara efisien. Editor ini banyak dipilih oleh pengembang karena kemampuannya dalam meningkatkan produktivitas, kemudahan kustomisasi, serta stabilitas performa dalam mengelola proyek skala kecil hingga besar. Dalam praktik pengembangan perangkat lunak, VS Code terbukti mampu mempercepat proses coding dan debugging berkat ekosistem plugin yang luas dan antarmuka yang intuitif (Bihesa et al., 2022).
